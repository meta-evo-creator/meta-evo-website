## 🔧 技术配置

### **GitHub Pages配置**
```yaml
# 仓库设置 → Pages
Source: Deploy from a branch
Branch: main
Folder: / (root)
Custom domain: meta-evo.com (可选)
Enforce HTTPS: ✅ 启用
```

### **自定义域名配置**
```
1. 在域名注册商处添加CNAME记录:
   Name: @
   Type: CNAME
   Value: username.github.io

2. 在GitHub仓库设置中添加自定义域名:
   Custom domain: meta-evo.com

3. 等待DNS传播（最长48小时）
```

### **HTTPS配置**
```
GitHub Pages自动提供HTTPS
无需额外配置
证书自动续期
```
