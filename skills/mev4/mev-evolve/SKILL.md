**mev-evolve** 是MEV4技能体系中的第四个核心技能，专注于实现从发现问题到持续成长的完整闭环进化。让AI助手具备持续学习和改进的能力，实现真正的智能进化。

- 提供经验复用工具

### 5. 标准化产品化
- 将成功经验标准化
- 建立标准化流程和规范
- 产品化可复用解决方案
- 提供标准化工具和模板

### 6. 推广应用扩展
- 推广成功经验和最佳实践
- 扩展到其他场景和应用
- 建立经验分享机制
- 促进知识传播和应用

### 7. 持续成长循环
- 建立持续改进循环机制
- 监控成长进度和效果
- 调整优化成长策略
- 实现持续能力提升

## 🔧 技术实现

### 技能配置
```yaml
skill:
  name: mev-evolve
  version: 1.0.0
  description: 持续进化与闭环成长技能
  author: MetaEvo
  category: evolution
  tags: [evolution, learning, improvement, growth]
  
  capabilities:
    - problem-identification
    - analysis-solution
    - effect-evaluation
    - experience-encapsulation
    - standardization-productization
    - promotion-extension
    - continuous-growth
```

### 使用示例
```javascript
// 应用mev-evolve技能进行经验学习
const mevEvolve = require('mev-evolve');

async function learnFromExperience(experienceData) {
  // 1. 问题识别和分析
  const problemAnalysis = await mevEvolve.problemIdentification({
    experience: experienceData.experience,
    context: experienceData.context,
    outcomes: experienceData.outcomes
  });
  
  // 2. 解决方案和实施
  const solution = await mevEvolve.analysisSolution({
    problem: problemAnalysis.problem,
    rootCause: problemAnalysis.rootCause,
    constraints: experienceData.constraints
  });
  
  // 3. 效果评估和度量
  const effectEvaluation = await mevEvolve.effectEvaluation({
    solution: solution,
    implementation: experienceData.implementation,
    metrics: experienceData.metrics
  });
  
  // 4. 经验封装和复用
  const encapsulatedExperience = mevEvolve.experienceEncapsulation({
    problem: problemAnalysis,
    solution: solution,
    evaluation: effectEvaluation,
    learnings: experienceData.learnings
  });
  
  return encapsulatedExperience;
}
```
