```
1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 仓库名称: meta-evo-website
4. 描述: MetaEvo official website
5. 公开仓库
6. 不初始化 README
7. 点击 "Create repository"
```

### **步骤2：推送代码（您操作）**
```
1. 获取仓库URL: https://github.com/your-username/meta-evo-website.git
2. 在本地执行:
   git init
   git add .
   git commit -m "Initial commit: MetaEvo website"
   git branch -M main
   git remote add origin https://github.com/your-username/meta-evo-website.git
   git push -u origin main
```

### **步骤3：启用 GitHub Pages（您操作）**
```
1. 进入仓库 Settings → Pages
2. Source: Deploy from a branch
3. Branch: main, folder: / (root)
4. 点击 Save
5. 等待部署完成（约1-2分钟）
```

### **步骤4：配置自定义域名（您操作）**
```
1. 在 Pages 设置中添加自定义域名: meta-evo.com
2. 在域名注册商处添加 CNAME 记录:
   Name: @
   Type: CNAME
   Value: your-username.github.io
   TTL: 3600
3. 等待 DNS 传播（最长48小时）
```
